# headlessui-float.vercel.app

Documation for [Headless UI Float](https://github.com/ycs77/headlessui-float).

## Contributing

This site is built with [VitePress](https://vitepress.vuejs.org/) and depends on [@vue/theme](https://github.com/vuejs/theme). Site content is written in Markdown format located in src. For simple edits, you can directly edit the file on GitHub and generate a Pull Request.

For local development, yarn is preferred as package manager:

```
yarn
yarn dev
```
